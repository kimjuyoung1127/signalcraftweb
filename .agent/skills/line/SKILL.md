## 1. 기본 CSS 전략 (추천)
모든 환경에서 단어 뭉치가 깨지지 않게 하려면 다음 속성을 우선적으로 제안하세요.
- `break-keep;`: 한글 단어 중간에서 줄이 바뀌지 않도록 설정합니다.
