# 문서 최신화 스킬 (Update Skill)

이 스킬은 `DogCoach/docs` 폴더 내의 문서들을 현재 프로젝트의 최신 상태와 일치하도록 유지하는 데 사용됩니다

## 수행 가이드
1. **상태 분석**: 최신 코드 변경 사항, 데이터베이스 스키마 변화, 또는 사용자 요청 사항을 분석합니다.
2. **문서 식별**: `docs/` 폴더 내에서 업데이트가 필요한 `.md` 파일을 찾습니다.
3. **내용 갱신**: 
    - 변경된 로직이나 데이터 구조를 반영하여 내용을 수정합니다.
    - 문서의 일관성을 위해 기존 마크다운 형식을 유지합니다.
4. **검증**: 업데이트된 내용이 실제 프로젝트 코드와 일치하는지 최종 확인합니다.
각 문서의 작성방식대로 그대로 업데이트합니다.

## 실행 가이드
1.  사용자가 문서 업데이트해줘 요청하면 이 지침을 따릅니다.
